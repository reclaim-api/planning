Common URL Model
========

There are only two parent orgs currently.

* {parentorg} = reclaimhosting.com
* {parentorg} = stack.network

**Reclaim Core**
* org.{parentorg}
* user.{parentorg}
* domain.{parentorg}
* app.{parentorg}
* instance.{parentorg}
* server.{parentorg}

**Each App On Network Gets Root & API**
* root: {sub}.{sld}.{tld}.{geo}.{app}.{parentorg}
* api: {sub}.{sld}.{tld}.{geo}.{app}.api.{parentorg}
